# TJA Decomposition Tool

段位認定などの連結された `.tja` ファイルを、1曲ごとのファイルに分割・抽出するWebツールです。

👉 **[ツールを開く](https://fjfjdkdk706-lgtm.github.io/TJA-Decomposition-tool/)**

## 使い方

1. **ファイルを選択**: 「📂 ファイルを選択」ボタンから `.tja` ファイルを選びます。
2. **ファイルを分解**: ファイル名が表示されたらボタンを押します。
3. **自動ダウンロード**: 分割されたファイルが入ったZIPファイルが自動的にダウンロードされます。

## 特徴

* 🛠️ **インストール不要**: ブラウザですぐに動きます。
* 🔒 **安全**: ファイルはサーバーに送信されず、すべてブラウザ内で完結して処理されます。
* 📄 **文字コード対応**: UTF-8 と Shift-JIS に対応。
